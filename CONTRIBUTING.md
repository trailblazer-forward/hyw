# 关于贡献

如果你想要修改某一处你发现的错误，或是创建一个新的页面，你可以按照如下步骤对本仓库进行贡献。

在参与贡献之前，你需要学会 [Git](https://github.com/git-guides) 和 [GitHub](https://docs.github.com/zh/get-started/start-your-journey/hello-world) 的基本使用。

1. [fork](https://docs.github.com/zh/pull-requests/collaborating-with-pull-requests/working-with-forks/fork-a-repo) 本仓库。
2. 在你创建分支中作出修改，并创建 [commit](https://github.com/git-guides/git-commit)。
3. 发起一个 [Pull Request](https://docs.github.com/zh/pull-requests/collaborating-with-pull-requests/proposing-changes-to-your-work-with-pull-requests/creating-a-pull-request)，并等待审核。
4. 如果审核通过，恭喜你成功为 HYW-wiki 作出了贡献，现在你可以在网站上看到你做出的修改了！
5. 如果没有审核通过，通常审核人员会留下修改意见，按照意见修改完毕之后再次等待审核即可。
